python中的[i:j:k]
2023-11-22|技术
字数总计: 92|阅读时长: 1 分钟

对于一维数组
a = [0, 1, 2, 3, 4, 5]
a[i:j:k]

  • 正序(k>0)
    i起始下标，j表示终止下标-1(下标0开始),k是步长，默认为1

  • 倒序(k<0)
    i起始下标，j表示终止下标+1,
